Gambaran Umum

Putri Minesweeper adalah game puzzle sederhana yang dirancang untuk dimainkan langsung di perangkat Anda. Berdasarkan implementasi aplikasi saat ini, aplikasi berfokus pada permainan lokal dan penyesuaian pengaturan permainan.

Putri Minesweeper tidak memerlukan pendaftaran akun dan tidak meminta pengguna mengirimkan data pribadi untuk dapat bermain.

Informasi yang Kami Kumpulkan

Putri Minesweeper tidak secara sengaja mengumpulkan informasi pribadi seperti nama lengkap, nomor telepon, alamat, atau informasi pembayaran melalui aplikasi ini.

Aplikasi dapat menyimpan preferensi dasar yang berkaitan dengan permainan di perangkat Anda, seperti pengaturan game lokal, untuk mendukung pengalaman penggunaan aplikasi. Informasi ini tetap berada di perangkat kecuali Anda menghapus aplikasi atau membersihkan data aplikasi.

Penggunaan Informasi

Setiap data lokal aplikasi hanya digunakan untuk mendukung fungsi inti aplikasi, seperti:

  • menyimpan konfigurasi permainan seperti ukuran papan dan jumlah ranjau,
  • mendukung alur permainan dan pengalaman pengguna,
  • menjaga aplikasi tetap berfungsi dengan baik di perangkat Anda.

Penyimpanan dan Penghapusan Data

Putri Minesweeper saat ini tidak membuat akun pengguna dan tidak menggunakan layanan backend untuk data permainan. Berdasarkan implementasi aplikasi saat ini, preferensi permainan dan data serupa tetap berada di perangkat Anda sampai Anda menghapus aplikasi atau membersihkan penyimpanan lokal aplikasi melalui pengaturan perangkat.

Karena versi aplikasi saat ini tidak menyimpan data pribadi berbasis akun di server, saat ini tidak ada proses penghapusan akun atau data server secara terpisah.

Pembagian Data

Putri Minesweeper tidak menjual informasi pribadi dan tidak secara sengaja membagikan informasi pribadi kepada pihak ketiga melalui fungsi inti permainan aplikasi ini.

Jika pada versi mendatang aplikasi menambahkan analitik, iklan, penyimpanan cloud, atau sistem akun, Kebijakan Privasi ini akan diperbarui sebelum fitur tersebut dirilis.

Privasi Anak

Putri Minesweeper adalah game untuk khalayak umum. Aplikasi ini tidak dirancang untuk secara sadar mengumpulkan data pribadi dari anak-anak. Jika Anda meyakini seorang anak telah memberikan informasi pribadi melalui versi aplikasi di masa mendatang, silakan hubungi pemilik aplikasi agar tindakan yang sesuai dapat dilakukan.

Keamanan

Upaya yang wajar dilakukan untuk menjaga implementasi aplikasi tetap sederhana dan meminimalkan pemrosesan data yang tidak diperlukan. Karena tidak ada sistem yang dapat menjamin keamanan absolut, pengguna juga disarankan menjaga perangkatnya tetap diperbarui dan terlindungi.

Pada versi saat ini, aplikasi dirancang untuk permainan lokal dan tidak bergantung pada login akun pengguna atau penyimpanan jarak jauh untuk penggunaan inti. Jika versi mendatang memperkenalkan fitur berbasis jaringan atau layanan pihak ketiga, Kebijakan Privasi ini akan diperbarui sebelum perubahan tersebut dirilis.

Perubahan Kebijakan Privasi

Kebijakan Privasi ini dapat diperbarui dari waktu ke waktu untuk mencerminkan perubahan aplikasi, kebutuhan hukum, atau pembaruan operasional. Versi terbaru yang dipublikasikan pada halaman ini harus dianggap sebagai versi yang berlaku.
